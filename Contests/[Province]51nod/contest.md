# 智商对决
## 题目限制
$2000$ms $512$M

## 题目描述
在一场脑力竞赛中，小T突破重重阻碍，披荆斩棘，终于闯到了最终PK环节，对战上个赛季的冠军A sir。 
已知比赛规则如下：

1. 比赛共有$n$项能力测试，每项能力测试将有一个重要度，第$i$项测试的重要度$c_i$由小T决定，但要求 $l_i<=c_i<=r_i$
2. 在第$i$项能力测试中，各选手可以获得的最终收益为$c_i \times 本项能力测试得分$，每项能力测试满分为$X$
3. 若小T在$n$项能力测试中的总收益$>=$A sir的总收益，小T将获得本场比赛的冠军

通过占卜，小T已知A sir将在每场能力测试中的得分，A sir第i项能力测试得分将为$b_i$，而小T目前在这$n$项能力测试中的得分均为$0$。但小T正是因其强大的学习能力才冲进决赛，因此，小T可以花费$1$小时的学习时间来让自己在第$i$项能力测试中多得$1$分，但无论如何学习得分都不会超过满分$X$。  
时间紧迫，小T想知道自己最少学习多少小时可以赢得PK比赛。

## 输入格式
第一行两个正整数$n$,$X$  
接下来$n$行，每行三个正整数$b_i$ $l_i$ $r_i$，如题所述

## 输出格式
一行一个整数表示答案

## 数据范围
对于20%的数据: $1<=n<=8,1<=X<=8,l_i=r_i$  
对于40%的数据: $1<=n<=1000,1<=X<=100000,l_i=r_i$  
对于60%的数据: $1<=n<=100000,1<=X<=100000,l_i=r_i$  
对于100%的数据: $1<=n<=100000,1<=X<=100000,1<=l_i<=r_i<=100000,0<=b_i<=X$  

## 样例

### Case1

#### 输入

```
2 100
85 2 3

```

#### 输出

```
115
```

### Case2

#### 输入

```
2 100

```

#### 输出

```
77
```

