# 是图论还是字符串
## 题目限制
$2000$ ms $256$ M

## 题目描述
题目很简单，给出一个长度为$n$只包含小写字母`a,b,c,d,g,e,f,g,h`的字符串$s$，根据字符串$s$建立一张无向图：

若$|i-j| = 1$，则在图中给$i,j$连边。
若$s_i = s_j$，在图中给$i,j$连边。

求这张图的直径，即求$max(dis[i][j])$($1 \leq i,j \leq n$),以及距离等于直径的点对(无序点对)个数。
$dis[i][j]$指$i$到$j$的最短距离。

## 输入格式
第一行一个正整数$n$，表示字符串长度 
第二行一个字符串$s$

## 输出格式
一行两个整数，分别表示图的直径和距离等于直径的点对数量

## 数据范围
对于20%的数据： $n \leq 300$ 
对于40%的数据： $n \leq 3000$ 
对于另外20%的数据： 字符串中只包含小写字母`a,b` 
对于100%的数据: $n \leq 100000$

## 样例

### Case1
#### 输入
```
15
efddbbggghaaccc 
```
#### 输出
```
12 2
```

### Case2
#### 输入
```
3
abc
```
#### 输出
```

```

### Case 3
#### 输入
