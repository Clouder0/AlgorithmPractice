# Title: 乌鹊南飞(*Flying*)

# Backgroud

>**“月明星稀，乌鹊南飞，此非曹孟德之诗乎？”**  

PF在操场上仰天长太息，天朗气清，惠风和畅，蓝天白云，乌鹊五里一徘徊。  
聪明的PF觉得乌鹊是个路痴，并且给出了严密的证明：

>**绕树三匝，何枝可依？**  

于是富有同情心的PF小朋友决定帮助乌鹊择木而栖。  
他对着黑不溜秋的乌鹊大喊：

>**“彼且奚适也？”**  

乌鹊答曰：  

>**吾有血肉翅，逼迫兼老鹰。**  
>**欲得石子尽，方能饮水回。**  
>**路且遥且险，鹰且凶且恶。**  
>**恶鹰有其重，重辄千百斤。**  
>**欲过鹰不重，且需路最短。**  
>**然石不可缺，尔有何良方？**  

聪明的PF被鸟语搞得惘然失措，而乌鹊徘徊不肯去，你能帮助可怜的PF解决这个问题，让他免于成为**普罗米修斯二代**吗？  

走运的是，你已经**鸟语三千八百九十二级**了，你看乌鹊的话就像看母语一样自然简单。

# Description

有$n$个点，$m$条边的无向图。  
每条边有两个权值：$dis$ 和 $weight$，分别代表这条边的长度和在这条边上的老鹰的重量。  
有$k$个石子散落在点上，现在，你需要：  

找出一条路径，  

1. 起点为$s$，终点为$t$。
2. 保证经过的所有边的$weight$不超过$limit$，且$limit$最小。  
2. **所有有石子的点都要经过**。  

满足上述条件的同时，使得路程$dis$最小。

# Input Format

第一行输入$5$个整数，点的数量$n$，便的数量$m$，石子的数量$k$，起点$s$，终点$t$。  
第二行输入$k$个整数，第$i$个整数记为$pos_i$，代表第$i$个石子在点$pos_i$上。  

随后$m$行，每行输入$4$个整数，$x_i$，$y_i$，$dis_i$和$weight_i$，  
代表有一条连接$x_i$和$y_i$的双向边，$dis$值为$dis_i$，此边上栖息的老鹰重量为$weight_i$。  
若$weight_i == 0$则代表此边上没有老鹰。

# Output Format

两个整数，分别代表求得的$limit$和满足题意的路径的所有边的$dis$和。  
若找不到方案，则输出一个整数：$-1$

# Data Range

对于所有数据，保证$1 ≤ n ≤ 10^5$,$1 ≤ m ≤ 10^6$,$1 ≤ s,t,x_i,y_i,pos_i ≤ n$，$0 ≤ dis_i,weight_i ≤ 10^9$,$1 ≤ k ≤ 10$。  
保证$x_i ≠ y_i$，$s ≠ t$。
