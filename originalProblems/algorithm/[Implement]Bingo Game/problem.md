# Title: Bingo游戏(BingoGame)

# Backgroud

在某个不知名的公司的春节晚会上，开展了一个奇怪的游戏，叫做**BingoGame**。  
每个人可以领到一张牌，牌上画了一个矩阵，写了互不相同的数字。  
经过了冗长无聊的领导讲话后，主持人开始主持游戏了：他先告诉你游戏的范围，再念出一系列数字，直到有人**Bingo**为止。  
什么叫游戏的范围呢？就是指矩阵的哪些位置是有效的。  
在范围中的数字如果被念到，则可做上标记。当范围内所有数字都打上标记后，就可以喊出**Bingo**取得游戏的胜利。  
小K就是这个愚蠢的公司的员工，现在他很懒，不想玩这个愚蠢的游戏。你可以写一个程序来帮助他吗？

# Description

有一个$n \times m$的矩阵，矩阵上的每个位置都有一个互不相同的数字$k$。   
进行$T$轮游戏，每轮游戏会给出$k$个数字，以及该轮游戏的范围。请你判断小K能否在该轮游戏中**Bingo**。  
由于晚会要消磨时间，所以$T$可能很大。为了增加游戏难度，所以$n$与$m$可能很大。

# Input Format

第一行，给出$n$，$m$和$T$。
接下来输入一个$n \times m$的矩阵，代表矩阵上的数。
然后进行$T$轮游戏：
每轮游戏第一行给出数字个数$k$。
接着给出一个$n \times m$的矩阵，每个位置的值只有$0$和$1$，其中$0$代表该位置不属于该轮游戏范围，$1$代表该位置属于该轮游戏范围。
随后按顺序给出$k$个数字，代表主持人念出的数字。

# Output Format

输出$T$行。
每轮游戏，若小K能够**Bingo**则输出`Bingo!`，否则输出`No!`。
